# Lab Studies Manager
## CS5610 Project 3 Lab Studies Manager

The Lab Studies Manager is built with **Node**, **Express**, **MongoDB**, and **React** frontend.
The Lab Studies Manager is a one stop website to manage your participants list and scheduling web app to be able to conveniently check everything you need instead of managing an unweildy spreadsheet.
The live production link is [here](https://projectfinal.timyim.info/).
It has 1000+ synthetic records, allows full CRUD operations on participants, studies, and sessions.

## How to Build & Run Locally

### 1. Clone the Repository
```bash
git clone https://github.com/airyimbin/lab_studies_manager.git
cd lab_studies_manager
```

### 2. Install Dependencies
```bash
cd frontend/
npm install
cd backend/
npm install
```

### 3. Create .env file for backend
```bash
JWT_SECRET=secret
```

### 4. Docker Compose
```bash
docker compose build
docker compose up -d --build
```
Then open http://localhost:5173
